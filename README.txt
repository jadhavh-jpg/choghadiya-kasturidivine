Numerology with Hari - Quick package
====================================

Files:
- index.html   : Main single-file site (Choghadiya + Numerology + WhatsApp quick contact)
- style.css    : Optional CSS file (currently not used)
- README.txt   : This file

How to use:
1. Unzip the package.
2. Place the files in your GitHub repository (root or in a folder).
3. Commit & push to GitHub.
4. Deploy on Vercel or GitHub Pages:
   - For GitHub Pages: enable Pages from repository settings.
   - For Vercel: create a new project pointing to the repo, it will auto-deploy.

Notes:
- The widget uses public APIs: Nominatim (OpenStreetMap) for geocoding and sunrise-sunset.org for sunrise/sunset times.
- For production with heavy traffic, consider using your own API keys (Mapbox/Google) and server-side timezone handling.
- WhatsApp quick-contact uses the number +91 9354370946 (as provided). Change in index.html if needed.
